## 🌈 What is LearnMap?

LearnMap is a **web application** that helps anyone create a **clear step-by-step learning roadmap** for any skill.

👉 Instead of asking:
> “What should I learn next?”

LearnMap gives you:
- A structured path  
- A clear starting point  
- A final goal  

💡 Think of it like a **Google Maps for learning** — but for skills.

## ⚙️ How is it Done?

LearnMap is built using:

- 🐍 Python (logic)
- 🌐 Streamlit (web interface)

### Process:

1. User enters:
   - Skill (e.g., AI, Web Development)
   - Level
   - Duration
   - Goal

2. The system:
   - Processes inputs
   - Structures a roadmap

3. Output:
   - Displayed as **visual cards**
   - Divided into phases
